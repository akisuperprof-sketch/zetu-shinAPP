# Brand Specification: 舌神 -ZETUSHIN- (v1.2)

## 1. コンセプト定義 (Concept)
「診断」ではなく「対話と導き」へ。
無機質なAIから、人格を持った体調の案内人「舌神（ぜつしん）」へのブランド転換。

- **名称**: 舌神（ぜつしん）
- **英語表記**: ZETUSHIN
- **役割**: ユーザーの舌から身体のコンディションを読み取る神様。
- **キャッチコピー**: 「舌からわかる 今日のコンディション」

## 3. UI/UX デザイン方針 (Research Minimal)
アニメ調のファンタジーUIから、清潔感のある「研究寄りミニマル」へ移行。

- **背景**: `#F8FAFC` (白に近い淡いグレー)
- **文字色**: `#1F3A5F` (信頼のネイビー)
- **アクセント**: `#6FC3B2` (癒やしのジェイド)
- **フォント**: `Noto Sans JP` (標準)

## 4. 舌神キャラクター (ZETUSHIN Character)
メインUIの邪魔をせず、案内役（ナビゲーター）として登場。

- **使用箇所**: スプラッシュ、撮影ガイド、結果画面のコメント。
- **役割**: ユーザーとの人格的な対話（AI → 舌神）。

## 5. コンディションマップ (Phase 1)
円形（Circular）のマップを採用。

- **横軸**: 寒 ← → 熱
- **縦軸**: 虚 (上) ← → 実 (下)
- **中央**: 安定バランス型（平和質）

## 7. 将来ロードマップ詳細 (Future Roadmap Details)

### Phase 3：コンディション軌跡 (Condition Trajectory)
- **内容**: 過去の診断結果（X, Y座標）を線で結び、体調の変化を可視化する。
- **目的**: 季節や生活習慣によるコンディションの揺らぎをユーザー自身が把握できるようにする。
- **UI**: XYマップ上に薄いグレーの線で過去1週間の動きを表示。

### Phase 4：気血水コンディションリング (Qi-Blood-Water Rings)
- **内容**: 気・血・水の充実度を3本のリングで表現する（Apple Watchのアクティビティリング形式）。
- **目的**: 漢方医学の根幹である「気血水」の状態を直感的に「満ちているか、欠けているか」で理解させる。
- **データ源**: 舌診解析の内部スコア（qi, blood, water）を反映。
