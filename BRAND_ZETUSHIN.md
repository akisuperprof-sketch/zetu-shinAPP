# Brand Specification: 舌神 -ZETUSHIN- (v1.2)

## 1. コンセプト定義 (Concept)
「診断」ではなく「対話と導き」へ。
無機質なAIから、人格を持った体調の案内人「舌神（ぜつしん）」へのブランド転換。

- **名称**: 舌神（ぜつしん）
- **英語表記**: ZETUSHIN
- **役割**: ユーザーの舌から身体のコンディションを読み取る神様。
- **キャッチコピー**: 「舌からわかる 今日のコンディション」

## 2. 開発ロードマップ (Roadmap)
| Phase | 内容 | ステータス |
| :--- | :--- | :--- |
| **Phase 1** | 9タイプコンディション分析・ブランド導入 | **実装中** |
| **Phase 2** | アルゴリズム強化（所見判定の精度向上） | 予定 |
| **Phase 3** | コンディション軌跡（過去の遷移可視化） | 予定 |
| **Phase 4** | 気血水コンディションリング（Apple Watch風UI） | 予定 |

## 3. UI/UX デザイン方針 (Research Minimal)
アニメ調のファンタジーUIから、清潔感のある「研究寄りミニマル」へ移行。

- **背景**: `#F8FAFC` (白に近い淡いグレー)
- **文字色**: `#1F3A5F` (信頼のネイビー)
- **アクセント**: `#6FC3B2` (癒やしのジェイド)
- **フォント**: `Noto Sans JP` (標準)

## 4. 舌神キャラクター (ZETUSHIN Character)
メインUIの邪魔をせず、案内役（ナビゲーター）として登場。

- **使用箇所**: スプラッシュ、撮影ガイド、結果画面のコメント。
- **役割**: ユーザーとの人格的な対話（AI → 舌神）。

## 5. コンディションマップ (Phase 1)
円形（Circular）のマップを採用。

- **横軸**: 寒 ← → 熱
- **縦軸**: 虚 (上) ← → 実 (下)
- **中央**: 安定バランス型（平和質）

## 6. データ構造（将来拡張用）
Phase 4 を見据え、気・血・水のデータを内部的に保持可能とする。
