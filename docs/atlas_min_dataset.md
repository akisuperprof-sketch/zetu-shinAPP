## 各項目の説明
*   **image_id**: 連番やユニークID
*   **提供同意**: 研究利用範囲 / 保存期間 / 二次利用可否 など
*   **撮影条件**: 光源（自然光/蛍光灯など） / 距離 / 角度 / 食後判定 など
*   **期待ラベル**: `tongue_color` / `coat_color` / `coat_thickness` / `moisture` / `tongue_form`
*   **期待pattern**: `JIKKAN` / `KYOKAN` / `JITSUNETSU` / `KYONETSU` / `HOLD`
*   **備考**: 紫/黒などのフラグ事象、矛盾、品質が微妙であるなどのメモ
