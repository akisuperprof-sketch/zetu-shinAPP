# ZETUSHIN 成長シミュレーション

## 成長指標の定義

### 1. 共有率 (Share Rate)
診断結果をSNSまたはチャットツールで共有するユーザーの割合。

### 2. 到達人数 (Reach)
1つの共有投稿がフォロワーやタイムライン上で閲覧される平均人数。

### 3. 招待CVR (Invite conversion)
共有された内容をクリックし、実際に診断を開始した人の割合。

### 4. K-factor
1人のユーザーが新たに連れてくる平均ユーザー数。
- **数式**: `k = 共有率 × 到達人数 × 招待CVR`
- **目標**: `k > 1.0` であれば広告費ゼロで指数関数的に成長する。

## 仮説レンジ

| 指標 | 保守的 (Base) | 楽観的 (Aggressive) |
|:---|:---|:---|
| 共有率 | 3% | 10% |
| 到達人数 | 50人 | 200人 |
| 招待CVR | 2% | 5% |
| **K-factor** | **0.03** | **1.0** |

## 10万人到達への試算

| フェーズ | 累計ユーザー | 新規流入(自然) |
|:---|:---:|:---:|
| 研究β | 300 | - |
| 初期研究 | 3,000 | 90 |
| SNS拡散開始 | 10,000 | 1,000 |
| 加速フェーズ | 100,000 | 10,000/月 |

## 継続率の意義
成長したユーザーが週1回、月1回と継続することで、研究データ（経時変化）の価値は指数関数的に高まる。
継続率向上のために「履歴比較」と「季節レポート」を強化する。
