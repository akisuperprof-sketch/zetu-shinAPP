# ZETUSHIN 研究データ価値設計

## データ構造と匿名性の両立
ZETUSHINが収集するデータは、個人の特定を不可能にしつつ、学術的・臨床的な価値を最大化する設計となっている。

### 収集項目の価値
1. **舌画像**: 東洋医学的診断（舌診）の客観的評価基準の作成。AI学習用ラベル。
2. **問診回答**: 自覚症状と客観情報の相関分析。
3. **解析結果**: AIモデルの精度検証と、診断の標準化。
4. **経時変化**: 個人の健康推移、季節変調の統計解析。

## データ提供とエコシステム
集積されたデータは以下の形態で提供可能とする。

- **提供先**:
    - 全国の大学・医学部（東洋医学講座等）
    - 漢方メーカー・製薬会社（商品開発）
    - 基幹病院・クリニック（電子カルテ連携）

- **提供形式**:
    - **匿名統計データ**: 地域別、年齢別、体質別のマクロ分析結果。
    - **匿名画像セット**: 特定の研究目的（例：糖尿病と舌診の相関）に基づき、厳格に匿名化された学習用データ。

## 倫理的配慮と法的遵守
1. **匿名化の物理保証**: アップロード前にクライアント側でEXIF除去（GPS/デバイス情報）。
2. **研究同意の明確化**: 同意なきデータ収集の禁止。
3. **撤回の権利**: 同意の随時撤回機能の提供。
4. **透明性**: データの用途をLPおよびアプリ内で常に明示する。
