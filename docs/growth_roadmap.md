# ZETUSHIN 10万人拡散ロードマップ

## フェーズ概要
ZETUSHINは「研究的価値」と「ユーザーの自己理解」を両輪として、以下の5段階で拡大する。

### P0: 研究基盤 (Research Foundation)
- **目的**: 診断精度とプライバシー保護の物理的証明
- **KPI**: 研究同意率 90%以上、画像品質合格率 80%以上
- **施策**: 
    - 匿名ID (`anon_id`) によるデータ分離
    - クライアントサイドEXIF除去
    - 大学・研究機関との連携スキーム固定

### P1: 研究拡大 (Scaling Participation)
- **目的**: AI学習に必要な統計的データ数の確保
- **KPI**: サンプル数 3,000件突破
- **施策**:
    - 特定コミュニティ（鍼灸院、ヨガ教室等）へのクローズド導入
    - 研究参加バッジの付与

### P2: 拡散準備 (Viral Preparation)
- **目的**: SNS共有時の「自己理解」価値の最大化
- **KPI**: K-factor 0.1 達成準備
- **施策**:
    - 共有カード生成エンジンの実装（Feature Flag封印）
    - 体質チャートの視覚的リファイン

### P3: 小規模公開 (Soft Launch)
- **目的**: 拡散サイクルの検証と負荷分散
- **KPI**: K-factor 0.5 突破
- **施策**:
    - SNS共有機能の段階的解放
    - インフルエンサー・医療従事者による共有促進

### P4: 拡散加速 (Hyper Growth)
- **目的**: 一般層への定着と10万人突破
- **KPI**: 累計ユーザー 100,000名
- **施策**:
    - 地域別体質マップの公開
    - 季節の体調変化に応じたプッシュ通知/レポート配信
